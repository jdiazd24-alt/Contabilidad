Contenido del Proyecto

Esta página fue diseñada considerando los siguientes aspectos clave:

  • Diseño Responsivo: Uso de Bootstrap 5 para adaptarse a móviles, tablets y desktops.
  • Navegación Intuitiva: Menú superior fijo y acordeón para contenido organizado.
  • Colores UMG: Azul principal (#063968) y rojo acento (#FF0000).
  • Tipografía: Fuentes legibles (Segoe UI, Arial) con jerarquía clara.
  • Accesibilidad: Contraste adecuado, iconos descriptivos y estructura semántica.
  • Interactividad: Acordeón colapsable para evitar sobrecarga visual.

Objetivo: Ofrecer una experiencia limpia, profesional y fácil de consultar.

Diferencias Clave
Tipo Enfoque Usuarios Normas
Financiera Externa, histórica Inversores, bancos NIIF, PCGA
Administrativa Interna, futura Gerencia No regulada
Costos Cálculo de costos Producción Interna
Objetivos Principales
  • Planeación estratégica
  • Control de operaciones
  • Toma de decisiones informadas
  • Maximización de rentabilidad

Importancia: Proporciona información oportuna y relevante para la gerencia, no limitada por normas externas.

1. Rentabilidad y Eficiencia
  • ROI: (Ganancia Neta / Inversión) × 100
  • ROE: (Ganancia Neta / Patrimonio Neto) × 100
2. Análisis de Flujo de Efectivo

Clasifica flujos en:

  • Operativas
  • De inversión
  • De financiamiento
3. Decisión Comprar o Producir

Se analizan costos relevantes (diferenciales). Ejemplo:

Costo de producir = $8/unidad
Costo de comprar = $10/unidad
→ Producir es mejor (si no hay restricciones)
4. Análisis Patrimonial
  • Razón de endeudamiento: Deuda Total / Patrimonio
  • Liquidez corriente: Activo Corriente / Pasivo Corriente
5. Proyecciones: VAN y TIR
VAN = Σ (Ft / (1+r)^t) - Inversión Inicial
Si VAN > 0 → Proyecto viable
TIR: Tasa donde VAN = 0
Si TIR > costo de capital → Aceptar
